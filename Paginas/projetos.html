Barbie Cards

Esse projeto de Site se trata de um exibidor de personagens, baseado na animação Barbie Live in The DreamHouse, o usuário deve escolher um personagem e assim que o cursor for posicionado em cima do personagem selecionado o site exibe a foto, nome e as informações referentes a personagem.

Cinéfilo Ou Não

Esse projeto de Site se trata de listas com dicas de filmes para assistir. Além da Homepage o site conta com uma lista dos 100 Filmes Que Todo Mundo Deveria Assistir Antes de Morrer e uma lista dos 25 Filmes Que Toda Criança Deveria Assistir com seus respectivos nome e sinopse.

RBD Quiz

Esse projeto de Site se trata de um Quiz o usuário deve ler e responder as quatro perguntas. Que são alternadas automaticamente e posteriormente o resultado é exibido. Esse Quiz foi desenvolvido com base nas letras das músicas do grupo musical RBD
